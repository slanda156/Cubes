<h2>Controls</h2>
<table>
    <tr>
        <td>WASD:</td>
        <td>Movement</td>
    </tr>
    <tr>
        <td>T:</td>
        <td>Skip Build Time</td>
    </tr>
    <tr>
        <td>F:</td>
        <td>Place Tower</td>
    </tr>
    <tr>
        <td>E:</td>
        <td>Use Equiped Item</td>
    </tr>
    <tr>
        <td>TAB:</td>
        <td>Switch Hud Modes</td>
    </tr>
    <tr>
        <td>Spacebar:</td>
        <td>Mind Transfer</td>
    </tr>
    <tr>
        <td>LMB:</td>
        <td>Shoot</td>
    </tr>
    <tr>
        <td>RMB:</td>
        <td>Teleport</td>
    </tr>
</table>
<h3>Hudmodes:</h3>
<ol>

<li>Healthbar</li>
<li>2. & Level, Reload, Resources, Status Icons, Equiped Item</li>
<li>3. & XP, Status Text, Inventory</li>
