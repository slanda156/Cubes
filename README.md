<h2>Controls</h2>
<table>
    <tr>
        <td>WASD:</td>
        <td>Movement</td>
    </tr>
    <tr>
        <td>T:</td>
        <td>Skip build time</td>
    </tr>
    <tr>
        <td>F:</td>
        <td>Place tower</td>
    </tr>
    <tr>
        <td>E:</td>
        <td>Use equiped item</td>
    </tr>
    <tr>
        <td>TAB:</td>
        <td>Switch hud modes</td>
    </tr>
    <tr>
        <td>Spacebar:</td>
        <td>Mind transfer</td>
    </tr>
    <tr>
        <td>LMB:</td>
        <td>Shoot</td>
    </tr>
    <tr>
        <td>RMB:</td>
        <td>Teleport</td>
    </tr>
</table>
<h3>Hudmodes:</h3>
<ol>
<li>Nothing</li>
<li>Healthbar</li>
<li>2. & Level, reload, resources, status icons, equiped item</li>
<li>3. & XP, status text, inventory</li>
<li>4. & Performance</li>
</ol>
